Desigining a Hall effect sensor signal processing module

The brief of this group project was to use a 2 axis gantry to detect magnets within a sandpit, and then mark their location by placing a marker on top of each magnet.

My role within the group was electronics design, which meant designing and building the sensor module, as well as the motor controllers to control the gantry stepper motors.
